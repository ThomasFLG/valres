MES PROJETS

PROJET VALRES

Développement d'une application web en 2ème année de BTS

Une application web pour la structure La Maison des Ligues de Lorraine (M2L) qui offre des salles avec des équipements informatiques, de réunion ou d’amphithéâtre pour des associations sportives ou non.

Fonctionnalités principales :

  • Le client peut réserver une salle.
  • Le client peut voir ses réservations et les modifier.
  • La secrétaire peut valider des réservations.
  • L’administrateur inscrit les utilisateurs et gère leurs droits.

Technologies utilisées :

  • HTML, CSS, JavaScript
  • PHP, MySQL
  • Framework Bootstrap
Voir le code sur GitHub Voir la démo en ligne

Retour d'expérience :

Ce projet m'a permis de développer mes compétences en gestion de base de données et en développement d'applications web avec une approche centrée sur l'utilisateur.

PROJET STORMSHIELD

renforcement du firewall de l'Hopital Joseph Ducuing

Mise en place de règles de filtrage ACL (Access Control List), qui filtre les paquets des données qui entrent et sortent de l'hôpital. Le tri se fait sur plusieurs critères, l'origine du paquet, sa destination, son port d'origine et son port de destination.
